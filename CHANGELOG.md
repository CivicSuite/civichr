# Changelog

All notable changes to CivicHR are documented here. This project follows Keep a Changelog style and uses semantic versioning.

## [0.1.0] - 2026-04-27

### Added

- CivicHR package and FastAPI runtime foundation.
- Personnel-policy lookup outlines.
- Handbook summary drafts.
- Job-description draft scaffolds.
- Classification and salary schedule reference lookup.
- Onboarding and training checklist support.
- Grievance/complaint intake templates only.
- HR source-readiness review and sensitive-topic preflight.
- Browser-verified public UI and docs landing page.
- Release gates for docs, tests, Ruff, build artifacts, package metadata, and placeholder imports.

### Not Shipped

- HRIS, payroll, benefits administration, or personnel records management.
- Employment-law advice or autonomous HR determinations.
- Personnel-file ingestion or PHI storage.
- Live LLM calls or external HR/payroll connector runtime.
