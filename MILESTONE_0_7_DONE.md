# CivicHR v0.1.0 Foundation Complete

CivicHR v0.1.0 ships the module foundation for HR policy support: source review, sensitive-topic preflight, policy lookup outlines, handbook summaries, job-description drafts, classification references, onboarding/training checklists, intake templates, local FastAPI runtime, documentation, tests, and release automation.

Boundaries: no HRIS, payroll, benefits administration, personnel records management, employment-law advice, autonomous HR decisions, case tracking, personnel-file ingestion, PHI storage, live LLM calls, or external HR/payroll connector runtime.
