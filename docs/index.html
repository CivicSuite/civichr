CivicSuite module

CivicHR

HR policy support for municipalities: policy lookup outlines, handbook summaries, job-description drafts, optional job/onboarding workpapers, onboarding/training checklists, and sensitive-topic preflight.

v0.1.1 shipping

For staff

Use CivicHR to prepare drafts from HR-approved sources. HR and counsel still decide and approve.

For IT

Install locally with python -m pip install -e ".[dev]", run python -m uvicorn civichr.main:app, and set CIVICHR_WORKPAPER_DB_URL for optional job/onboarding persistence.

Boundaries

No HRIS, payroll, benefits administration, personnel records management, employment-law advice, personnel-file ingestion, or live LLM calls ship in v0.1.1.

Repository

CivicSuite/civichr

Dependency

Pinned to civiccore==0.3.0. CivicCore remains dependency-only; it never imports CivicHR.
