CivicHR User Manual

For staff: CivicHR prepares HR policy lookup outlines, handbook summaries, job-description drafts, onboarding packets, training checklists, and intake templates. HR and counsel still approve all sensitive work.

For IT: install with python -m pip install -e ".[dev]" and run with python -m uvicorn civichr.main:app --host 127.0.0.1 --port 8138. The package depends on civiccore==0.2.0 and runs locally.
