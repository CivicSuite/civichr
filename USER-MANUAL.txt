CivicHR User Manual

For staff: CivicHR prepares HR policy lookup outlines, handbook summaries, job-description drafts, onboarding packets, training checklists, intake templates, and optionally retrieves saved job/onboarding workpapers. HR and counsel still approve all sensitive work.

For IT: install with python -m pip install -e ".[dev]" and run with python -m uvicorn civichr.main:app --host 127.0.0.1 --port 8138. The package depends on civiccore==0.3.0 and runs locally. Set CIVICHR_WORKPAPER_DB_URL to enable SQLAlchemy-backed job-description and onboarding-packet records; leave it unset for deterministic stateless operation.
